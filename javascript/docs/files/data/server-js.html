amigo.data.server

This package was automatically created during AmiGO 2 installation.

Purpose: Useful information about GO and the AmiGO installation.  Also serves as a repository and getter for web resources such as images.

NOTE: This file is generated dynamically at installation time.  Hard to work with unit tests--hope it’s not too bad.  Want to keep this real simple.
